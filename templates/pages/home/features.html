User Features
  • Quickly filter data using predefined filters
  • Use custom filters to create complex queries
  • Sort data using predefined columns
  • View data in a paginated list type table
  • Access detailed and related property tables
  • Create and edit data using a variety of field controls
  • Generate links to any item in any view to share with others. Example: #!pages/details/32
Developer Features
  • View, Edit, and Delete data from virtually any type of REST server or data store using CanJS's Can-Connect
  • Customize the display of each data view for a complete user experiencer
  • Format fields using custom functions to display images, links, etc.
  • Develop code in manageable modules using ES6 code. ES6 is transpiled on the fly using StealJS
  • Create super efficient, progressivly loaded bundles using StealJS's companion package, Steal Tools
